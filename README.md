# Guess The Number Game (Python)

This is a simple Python game where the computer randomly selects a number and the user has to guess it.

## Features
- Random number generation
- User input for guessing
- Hints if the guess is too high or too low
- Displays number of attempts

## Tech Used
- Python

## How to Run
Run the file using Python:

python guess_num.py
